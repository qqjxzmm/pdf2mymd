流程说明：

1. 输入：批量PDF文件
2. PDF转换：将每个PDF转换为高分辨率PNG图片
3. 文件组织：为每个PDF创建独立输出文件夹
4. 预处理：对每张图片进行文字区域检测和占比计算
5. 判断处理：根据文字区域占比决定是否进行OCR
6. OCR处理：对符合条件的图片进行文字识别
7. 输出：将识别结果保存为MD文件，尽可能保留原格式
这个流程可以帮助您有效处理那些混合了大量图片的PDF文档，避免对纯图片页面进行
不必要的OCR处理。
